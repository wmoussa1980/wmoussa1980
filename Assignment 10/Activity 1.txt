    Get Grade

Get Grade
    Parameters:
        None
    Process:
        Display prompt for grade
        Get grade
    Return Value:
        Grade
        
Calculate average
    Parameters:
        total
        count
    Process:
        Average=total/count
    Return Value:
        Average

Process grade input
    Parameters:
        total
        count
    Process:
        Initialize count
        Inialize total
        Loop while True
            Get Grade
            total = total + grade
            Increment count by one
            if grade<0 Break
    Return Value:
        total
        count
